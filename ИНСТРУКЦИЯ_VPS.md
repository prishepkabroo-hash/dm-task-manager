# Как развернуть таск-менеджер на сервере (VPS)

Пошаговая инструкция. Займёт 15-20 минут.

---

## ШАГ 1. Арендовать сервер

Зайди на один из сайтов:
- **Aéza** — https://aeza.net (от 300 руб/мес)
- **Timeweb** — https://timeweb.cloud (от 300 руб/мес)
- **Selectel** — https://selectel.ru (от 400 руб/мес)

Что выбрать при заказе:
- **ОС**: Ubuntu 22.04 или 24.04
- **Тариф**: самый дешёвый (1 CPU, 1 ГБ RAM хватит)
- **Локация**: Москва или ближайший город

После оплаты тебе дадут:
- **IP-адрес** сервера (например `185.100.50.25`)
- **Пароль** от root

Запиши их — они понадобятся дальше.

---

## ШАГ 2. Подключиться к серверу

Открой Terminal на маке и набери (замени IP на свой):

```
ssh root@185.100.50.25
```

Он спросит `Are you sure...?` — набери `yes` и нажми Enter.
Введи пароль (при вводе символы не видны — это нормально).

Ты внутри сервера.

---

## ШАГ 3. Загрузить файлы на сервер

Открой ВТОРОЕ окно Terminal (Cmd + N) и набери ЭТИ КОМАНДЫ (замени IP):

```
scp ~/Desktop/dm-task-manager/server.py root@185.100.50.25:/root/
scp -r ~/Desktop/dm-task-manager/templates root@185.100.50.25:/root/
scp -r ~/Desktop/dm-task-manager/static root@185.100.50.25:/root/
scp ~/Desktop/dm-task-manager/deploy.sh root@185.100.50.25:/root/
```

Каждый раз введи пароль когда попросит.

---

## ШАГ 4. Запустить установку

Вернись в ПЕРВОЕ окно Terminal (где подключён к серверу) и набери:

```
chmod +x /root/deploy.sh && bash /root/deploy.sh
```

Подожди 1-2 минуты. Увидишь сообщение «ГОТОВО!»

---

## ШАГ 5. Проверить

Открой Chrome и набери (замени на свой IP):

```
http://185.100.50.25:8080
```

Логин: `admin`
Пароль: `admin123`

---

## Готово!

Теперь таск-менеджер:
- ✅ Работает 24/7
- ✅ Доступен с любого устройства по IP
- ✅ Автоматически запускается после перезагрузки сервера
- ✅ Делает бэкап базы каждый день в 3:00

---

## Полезные команды (запускать на сервере)

Проверить статус:
```
systemctl status dm-tasks
```

Перезапустить:
```
systemctl restart dm-tasks
```

Посмотреть бэкапы:
```
ls -la /opt/dm-tasks/backups/
```

---

## (Бонус) Привязать домен

Если хочешь чтобы вместо IP был адрес типа tasks.dudarev-motorsport.ru:

1. Зайди к своему регистратору домена
2. Добавь A-запись: `tasks` → `IP твоего сервера`
3. Через 5-10 минут можно открывать `http://tasks.dudarev-motorsport.ru:8080`
