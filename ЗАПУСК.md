# Dudarev Motorsport — Таск-менеджер

## Быстрый запуск

### 1. Убедитесь, что установлен Python 3
```
python3 --version
```
Если нет — скачайте с https://python.org

### 2. Запустите сервер
Откройте терминал (или командную строку), перейдите в папку `dm-task-manager` и выполните:

```
cd dm-task-manager
python3 server.py
```

### 3. Откройте в браузере
```
http://localhost:8080
```

## Доступ по сети (для всей команды)

Чтобы другие сотрудники могли подключаться к таск-менеджеру со своих ноутбуков:

1. Узнайте IP-адрес компьютера, на котором запущен сервер
   - Windows: `ipconfig` (смотрите IPv4)
   - Mac/Linux: `ifconfig` или `ip addr`
2. Другие сотрудники открывают `http://ВАШ-IP:8080`

## Функции

- Канбан-доска (Новые → В работе → Готово)
- Список задач с фильтрами
- Приоритеты (высокий / средний / низкий)
- Дедлайны с индикацией просрочки
- Назначение на сотрудника или отдел
- Комментарии к задачам
- Голосовой ввод задач (Chrome)
- Регистрация и авторизация
- Все данные сохраняются в файле dm_tasks.db
